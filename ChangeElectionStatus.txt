API endpoint name: ChangeElectionStatus.php

POST parameters:
    postAuthKey: our auth key
    adminId: string id of admin
    type: type of election LS or VS
    electionId: int id of election
    newStatus: int new status to be updated
    

Response parameters:
    success: boolean
    validAuth: boolean
    validAdmin: boolean (false for invalid admin or not logged in)
    validType: boolean  (false for invalid type)
    validElection: boolean  (false for invalid electionId)
    validStatus: boolean (false for newStatus is not 1 greater than currentStatus and status changes from 2 to 3)
