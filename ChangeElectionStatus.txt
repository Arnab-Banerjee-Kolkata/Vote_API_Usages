API endpoint name: ChangeStatus.php

POST parameters:
    postAuthKey: our auth key
    adminId: string id of admin
    type: type of election LS or VS
    electionId: int id of election
    newStatus: int new status to be updated
    

Response parameters:
    success: boolean
    validAuth: boolean
    validAdmin: boolean
    validType: boolean
    validElection: boolean
    validStatus: boolean
