API endpoint name: CreatePhaseElection.php

POST parameters:
    postAuthKey: our auth key
    stateElectionId: Integer id of state election
    phaseCode: String code of phase
    boothId: String id of booth
    
    
    
Response parameters:
    success: boolean
    validAuth: boolean
    validBooth: boolean(false for invalid or logged out booth)
    validStateElection: boolean(false for invalid state election id)
    validPhaseCode: boolean(false for invalid phase code)
    validElection: boolean(false if election is already created)
    electionId: Integer id of created election(-1 for failure)
