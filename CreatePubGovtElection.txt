API endpoint name: CreatePubGovtElection.php

POST parameters: 
	postAuthKey: our auth key
	stateCode: String code of state
	phaseCode: String code of phase
	type: String type of election(LOK SABHA/ VIDHAN SABHA)
	startTime: String start time in "HH:MM:SS" in 24hr format
	startDate: String start date in "YYYY-MM-DD" format
	endTime: String end time in "HH:MM:SS" in 24hr format
	endDate: String end date in "YYYY-MM-DD" format



Response parameters:
	success: boolean 
	validAuth: boolean
	validTime: boolean(false if election cannot be scheduled in the given time)
	electionId: Integer id of created election(-1 on failure)
