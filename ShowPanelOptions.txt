API endpoint name: ShowPanelOptions.php

POST parameters:
  postAuthKey: our auth key
  boothId: booth id
  aadhaarNo: aadhaar of the voter
  electionId: country election id
  type: type of election LOK SABHA or VIDHAN SABHA
  
Response parameters:
  validAuth: boolean
  validBooth: boolean
  validAadhaar: boolean
  validElection: boolean
  validType: boolean
  validApproval: boolean
  success: boolean
  phaseElectionId: string
