API endpoint name: ShowPubGovtElectionDetails.php

POST parameters:
    postAuthKey: our auth key
    electionId: Integer id of election
    
    
Response parameters:
    success: boolean
    validAuth: boolean
    validElectionId: boolean(false for invalid election)
    stateName: String name of state of election
    phaseCode: String code of phase
    type: String type of election
    status: Integer status of election
    startDateTime: String start date time in format 'YYYY-MM-DD HH:MM:SS' in 24hr format
    endDateTime: String end date time in format 'YYYY-MM-DD HH:MM:SS' in 24hr format
