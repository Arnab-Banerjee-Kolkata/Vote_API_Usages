API endpoint name: ValidateCredentials.php

POST parameters:
        postAuthKey: our auth key
        boothId: String id of booth
        aadhaarNo: String aadhaar number of voter
        
        
        
Response parameters:
        success: boolean
        validAuth: boolean
        validBooth: boolean(false for invalid booth id)
        validAadhaar: boolean(false for invalid aadhaar number)
