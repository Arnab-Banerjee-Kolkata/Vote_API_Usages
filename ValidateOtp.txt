API endpoint name: ValidateOtp.php


POST parameters:
        postAuthKey: our auth key
        electionId: Integer id of election
        aadhaarNo: String aadhaar number
        category: String category(booth or voter)
        boothId: String id of booth
        otp: Integer OTP
        
       
For validating voter OTP from remote booth

Response parameters:
    success: boolean
    validAuth: boolean
    validStatus: boolean(false if voter has already voted)
    boothOtp: Integer OTP of booth(-1 in case of invalid voter OTP)
    
    
    
For validating voter OTP from remote booth

Response parameters:
    success: boolean
    validAuth: boolean
    validApproval: boolean(false if voter is not approved to vote)
